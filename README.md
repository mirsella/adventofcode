# adventofcode

My Advent of Code solutions in Rust, grouped by year.

## Layout

- `2023/`: Rust workspace with one crate per day.
- `2024/`: Rust workspace with one crate per day.
- `template/`: starter template for new days.

## Run a day

```bash
cd 2024/day1
cargo run
```

## Test a full year

```bash
cd 2024
cargo test
```
